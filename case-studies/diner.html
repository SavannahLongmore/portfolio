Coded Diner Website

Hand-coded responsive diner site with smooth interactions and mobile-first design

HTML CSS JavaScript Responsive

Overview

This project focused on translating clean design into functional code. I built the entire homepage of a fictional diner website from scratch using HTML, CSS, and JavaScript. The goal was a responsive, engaging site that works beautifully on desktop and mobile while demonstrating solid front-end skills.

My Role

UX/UI Designer & Front-End Developer

Tools

HTML, CSS, JavaScript, VS Code

Focus

Responsive layout, interactivity, performance

Key Features & Challenges

I solved several responsive and interactive challenges while keeping the code clean and performant.

Performance & Optimization

I optimized images and code to keep the site fast and smooth:

  • Compressed images → reduced total size significantly
  • Mobile-first CSS → faster load on phones
  • Minimal JavaScript → carousel and nav work smoothly
